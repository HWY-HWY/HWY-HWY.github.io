HTML设置边框

概要

HTML可以通过给相应的元素设置border对象来设置该元素的边界，从而使该元素包含在设置的边框中。

设置border样式


<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>边框</title>
        <style type="text/css">
            div{
                width: 300px;
                height: 300px;
                border: 10px solid rgb(255,0,0);border: 10px solid rgb(255,0,0);border: 10px solid rgb(255,0,0);border: 10px solid rgb(255,0,0);border: 10px solid rgb(255,0,0);border: 10px solid rgb(255,0,0);
            }
        </style>
    </head>
    <body>
        <div>
        </div>
        <!–
            boder的组成
            1、边框的粗细程度
            2、边框的表现形式
                solid  实线
                dashed 虚线
                dotted 点划线（这个属性不兼容，即不同浏览器显示是不一样的）
            3、边框的颜色
                red         颜色英文单词
               #1234                   颜色十六进制数
                rgb(0,0,0)    颜色的rgb表示法
                rgba         其中a为阿尔法通道的意思
            三个属性位置可调换
        –>
    </body>
</html>

其中border的三个属性其实可以分别写，用border-width: 1px;　　border-style:solid;　　border-color:red;，分别用来指定边框的像素、样式、颜色。
